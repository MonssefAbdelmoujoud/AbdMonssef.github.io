Welkom

Welkom op mijn portfolio!

Ik ben Monssef Abdelmoujoud, een afgestudeerde student van de graduaatopleiding Programmeren aan Vives Hogeschool in Kortrijk. Met deze website wil ik mezelf voorstellen aan de buitenwereld, werk zoeken, en vooral laten zien waartoe ik in staat ben.

Ik ben sociaal, leergierig, en ik hou van een goede uitdaging, het is precies dat wat me motiveert om te blijven groeien. De wereld van development is gigantisch, maar ik voel me er thuis. Vooral het werken met API’s vind ik ontzettend boeiend.

Mijn interesse in programmeren is ontstaan binnen mijn vriendengroep, die vol zit met developers. Aangestoken door hun enthousiasme begon ik via zelfstudie aan Python, waarna ik de overstap maakte naar C#. Sindsdien ben ik niet meer gestopt.

Onlangs heb ik mijn eindwerk afgerond voor het laatste jaar van mijn opleiding. Hoewel ik nog niet precies weet welke richting ik op lange termijn wil uitgaan, weet ik wél dat ik ervaring wil opdoen en een steeds betere programmeur wil worden.

Naast programmeren hou ik van een beetje ontspanning: ik speel graag het kaartenspel Yu-Gi-Oh, ben geïnteresseerd in auto’s, en help af en toe mijn vader in zijn garage. Die combinatie van techniek, creativiteit en plezier typeert wie ik ben.

Projecten

AI-powered FAQ Chatbot Widget

Deze chatbot widget werd ontwikkeld tijdens mijn stage bij Flagstone en biedt gebruikers een slimme manier om veelgestelde vragen (FAQ) te raadplegen of eigen vragen te stellen via een AI-assistent.

De front-end werd opgebouwd met HTML, CSS en JavaScript (zonder frameworks) en bevat een responsieve, zwevende interface met live zoekfiltering, dynamische DOM-updates en vloeiende overgangen.

De back-end is ontwikkeld in ASP.NET Core Web API, waarbij FAQ-data wordt opgehaald en gebruikersvragen doorgestuurd worden naar de OpenAI API voor AI-gegenereerde antwoorden. De widget leest gegevens uit een SQL Server-database, met Entity Framework Core als ORM-laag.

Markdown-opmaak wordt correct weergegeven in de chat, en het ontwerp houdt rekening met gebruiksvriendelijkheid op verschillende schermgroottes. Het project legt de nadruk op duidelijke scheiding tussen interface, logica en datalaag.

Gebruikte technologieën: HTML, CSS, JavaScript, ASP.NET, Entity Framework Core, SQL Server, OpenAI API, JSON + Rest API, Visual Studio (Code), Git
